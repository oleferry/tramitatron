# Kit de marca Tramitatrón — v0.1

Este paquete contiene una primera definición estratégica y visual de la marca. Está preparado para tomar decisiones, validar con administraciones y encargar el arte final; no sustituye una revisión registral ni una auditoría de accesibilidad final.

## Contenido

- `docs/TRAMITATRON_BRAND_CONTENT_BOOK_v0.1.docx`: manual editable.
- `docs/TRAMITATRON_BRAND_CONTENT_BOOK_v0.1.pdf`: versión de lectura y presentación.
- `brand_tokens.json`: colores, tipografías, claims y reglas reutilizables por diseño y desarrollo.
- `copy_bank.md`: banco de mensajes y microcopy para tótem, web y venta institucional.
- `logo_routes.md`: evaluación estratégica de cuatro rutas de logotipo.
- `assets/logos/*.svg`: bocetos vectoriales editables de las rutas propuestas.
- `assets/previews/*.png`: previsualizaciones para revisión rápida.

## Recomendación de dirección

Usar la ruta **A. Camino T** como base del símbolo y combinarla con el detalle verbal de la ruta **B. Acento que resuelve**. El resultado debería ser sencillo, institucional y humano; no debe parecer una mascota, un robot ni una aplicación financiera.

## Antes de publicar

1. Realizar búsqueda de anterioridades en OEPM/EUIPO y revisar clases relevantes.
2. Confirmar dominio y perfiles sociales.
3. Encargar refinado vectorial y pruebas de reducción/monocromo.
4. Validar contraste y legibilidad en el tótem real.
5. Revisar traducciones oficiales al valenciano con una persona cualificada.
6. Aprobar reglas de co-branding con la administración contratante.
